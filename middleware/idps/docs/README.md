# Documentation et Fichiers d'Exemple IDPS

Ce dossier contient :
- **Fichiers CSV d'exemple** : Exemples de fichiers IDPS pour tester le middleware
- **Documentation** : `Persistance_Logs_Biometrics.docx` - Document de spécification original

## Fichiers CSV d'Exemple

Les fichiers CSV présents sont des exemples pour tester le middleware :
- `IDPS-TG-EID-WO-BACKLOG-2025-11-11.csv` : Exemple d'événement workflow (backlog)
- `IDPS-TG-EID-WO-FINISH-2025-11-11.csv` : Exemple d'événement workflow (finish)
- `IDPS-TG-EID-QC-ERROR-2025-11-11.csv` : Exemple d'erreur de contrôle qualité
- `IDPS-TG-EID-PERSO-ERROR-2025-11-11.csv` : Exemple d'erreur de personnalisation
- `IDPS-TG-EID-SUP-ERROR-2025-11-11.csv` : Exemple d'erreur de supervision

Ces fichiers peuvent être utilisés pour tester le middleware en les plaçant dans le répertoire `input/`.
